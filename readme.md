

## Descripción del proyecto

El proyecto se basa en una página web donde dos usuarios pueden jugar una partida jcj.
En esta los dos jugadores participantes no podran comenzar otra partida. Un jugador que 
no ha comenzado partida, podra ser espectador de otra.
